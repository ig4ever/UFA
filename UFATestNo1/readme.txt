Cara menjalankan program menggunakan Node.js
1. Ubah direktori ke folder UFATest1 pada terminal
2. Jalankan command berikut :
- 'node soalA.js' untuk melihat jawaban dari soal no 1 bagian A
- 'node soalB.js' untuk melihat jawaban dari soal no 1 bagian B
- 'node soalC.js' untuk melihat jawaban dari soal no 1 bagian C
